## <a name="now-available-windows-7-year-2-esu-for-cloud-solution-provider-csp"></a><a name="6"></a>Nu kan du använda Windows 7 Year 2-ESU för Cloud Solution Provider (CSP)

### <a name="categories"></a>Kategorier

- Datum: 2020-06-29
- Typ: erbjudanden/marknader
- Status: är nu tillgängligt

### <a name="summary"></a>Sammanfattning

Priserna för ESU-priser (Extended Security updates) för Windows 7, år 2 finns nu i [resurs galleriet](https://partner.microsoft.com/resources/collection/windows-esu-csp#/).

### <a name="impacted-audience"></a>Förverkad mål grupp

Alla partner som deltar i programmet för Cloud Solution Provider (CSP)

Den effektiva perioden av ESU för Windows 7 år 1 upphör den 12 januari 2021. Vi vet att kunder börjar planera för nästa år.  

För kunder som förväntar sig att fortsätta använda Windows 7-enheter i 2021, gör vi priser för Windows 7 år 2-ESU som är tillgängliga för leverantörer av moln lösnings leverantörer (CSP) nu för att hjälpa kunderna med sitt Windows 10-beslut. Besök [resurs galleriet om du](https://partner.microsoft.com/resources/collection/windows-esu-csp#/) vill ha mer information.

## <a name="billing-frequency-default-changing-from-annual-to-monthly"></a><a name="4"></a>Standard ändring av fakturerings frekvens från årlig till månatlig

### <a name="categories"></a>Kategorier

- Datum: 2020-06-17
- Typ: funktioner
- Status: kommer snart

### <a name="summary"></a>Sammanfattning

Standardvärdet för fakturerings frekvensen ändras från årlig till månatlig i partner Centers köp upplevelse.

### <a name="impacted-audience"></a>Förverkad mål grupp

Leverantörer av moln lösnings leverantör (CSP) och indirekta leverantörer

### <a name="details"></a>Information

Inställningen fakturerings frekvens anger hur ofta partnern debiteras för de tjänster som de har köpt. Partner ser den här inställningen i **gransknings** steget vid köp i Partner Center-katalogen. Den här inställningen har standardvärdet **årsvis** sedan årlig fakturering infördes för licensbaserade tjänster. Baserat på feedback från partnern kommer Partner Center-teamet att ändra denna standard från den 17 juni från **årlig** till **månatlig** , så att den bättre anpassas till det som förväntas av partner. Partner kan ändra detta innan du skickar en transaktion på sidan Granska eller efter en transaktion från sidan Hantera prenumeration. Tänk på att om du bara har stöd för årlig fakturering fortsätter du med standardvärdet per år. Den här standard ändringen gäller bara för användar upplevelsen i Partner Center-portalen, det finns inga ändringar i hur API-funktionen i Partner Center används.

Vi lura den här ändringen minskar möjligheten för partner att av misstag köpa en årlig fakturerings prenumeration oavsiktligt.

## <a name="changes-to-microsoft-365-f1"></a><a name="3"></a>Ändringar i Microsoft 365 F1

### <a name="categories"></a>Kategorier

- Datum: 2020-06-15
- Typ: erbjudanden/marknader
- Status: är nu tillgängligt

### <a name="summary"></a>Sammanfattning

Microsoft 365 F1 kommer att uppdateras med nya tjänster. Ett nytt erbjudande kommer att lanseras för att ersätta föregående F1 den 15 juni.

### <a name="impacted-audience"></a>Förverkad mål grupp

Leverantörer av moln lösnings leverantör (CSP) och indirekta leverantörer

### <a name="details"></a>Information

Microsoft 365 F1 har varit tillgängligt i Partner Center sedan den 2020 april. Förbättringar görs genom att lägga till stöd för nya tjänster: funktionerna för Exchange kiosk och MS search. De nya tjänsterna kommer att tillhandahållas automatiskt på alla befintliga prenumerationer baserat på de ursprungliga F1-erbjudandena i slutet av juni. De ursprungliga F1-erbjudandena kommer att ersättas den 15 juni med nya F1-erbjudanden (nedan) för nya prenumerations inköp.

Ursprungliga F1-erbjudanden:

   |**Erbjudandets namn**|**Erbjudande-ID**|**Material-ID**|**Etablerings-ID**|
   |-------------------|:------|:------|:------|
   |Microsoft 365 F1|1ce4a4ba-65f7-49e7-bb28-d92d07d58b86|1PI-00005|M365_F1|
   |Microsoft 365 F1-utvärdering|dd97e36a-f0df-4a9a-b461-625420656511|1PI-00006|M365_F1|

Partner som vill köpa nya F1-prenumerationer bör använda de följande ID: na för erbjudandet från och med den 15 juni. Partners kan se dessa ID: n i Partner Center-katalogen. Partner som utnyttjar API: erna för partner Center kan även använda informationen nedan.

Nya F1-erbjudanden:

   |**Erbjudandets namn**|**Erbjudande-ID**|**Material-ID**|**Etablerings-ID**|
   |-------------------|:------|:------|:------|

   |Microsoft 365 F1-utvärdering|2e4e9424-06db-4d35-b2d8-3a1b0ef1dc63|1PI-00013|M365_F1_COMM|

Partner med den aktuella Microsoft 365 F1-produkten har ingen åtgärd. Service planer för Exchange kiosk och MS search kommer att fyllas i i slutet av juni på sina befintliga Service planer.

Partner som vill köpa Microsoft 365 F1 efter 15 juni bör använda den nya F1-informationen för att se till att de köper rätt erbjudande. Nya ID för F1-erbjudanden och annan information kommer att uppdateras i pris List uppdateringen juli Cloud Solution Provider (CSP) den 1 juli.
