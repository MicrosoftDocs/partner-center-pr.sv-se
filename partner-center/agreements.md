# <a name="microsoft-customer-agreements-by-region-and-language"></a>Microsofts kund avtal efter region och språk

**Gäller för**

- Cloud Solution Provider program partners

**Lämpliga roller**

- Global administratör
- Administratör för användar hantering
- Administratörs agent
- Försäljnings agent
- Support agent

Innan du kan placera en order för en kunds räkning måste kunden godkänna och underteckna Microsofts kund avtal.

>[!IMPORTANT]
> Du måste bekräfta att en kund har godkänt Microsofts kund avtal genom att ange information om den person som signerade det. Se [Bekräfta en kunds godkännande av Microsofts kund avtal](confirm-consent.md) för mer information.

Microsofts kund avtal är regions-specifika och tillgängliga på flera språk. Hitta kundens plats och önskat språk i någon av listorna nedan för att ladda ned rätt avtal.

Om du inte är säker på vilka regioner och marknader du kan sälja till hittar du ditt företags plats i den här listan: [Cloud Solution Provider program regionala marknader och valutor](regional-authorization-overview.md).
