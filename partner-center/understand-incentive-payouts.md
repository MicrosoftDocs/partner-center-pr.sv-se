# <a name="view-manage-and-export-your-incentives-program-details"></a>Visa, hantera och exportera information om stimulans programmet

**Gäller för**

- Partnercenter

**Lämpliga roller**

- Stimulans administratör
- Stimulans användare
- Incitament skrivskyddad visning
- Global administratör
- MPN-partner administratör

Använd dessa sidor om du vill visa information om dina tidigare, väntande och kommande betalningar och intäkter, status för dina stimulans program på varje plats och din fullständiga transaktions historik. Du kan också hämta eller exportera detaljerad information för alla transaktioner. Alla incitaments sidor visar belopp i amerikanska dollar.

>[!NOTE]
>Du ser bara data för MPN-ID: n och program som du är associerad med. Om du vill ha åtkomst till ytterligare data kan du läsa mer i stimulans administratören. 

## <a name="access-the-incentives-pages"></a>Öppna incitaments sidorna

Du kan få åtkomst till stimulans information på några sätt från Partner Center- [instrument panelen](https://partner.microsoft.com/dashboard):

- Välj **incitament** på menyn till vänster. Flera alternativ visas under **incitament** , inklusive **Översikt** . Utforska de här alternativen för att lära dig status för stimulans program, incitaments betalningar, intäkter och anspråk.
- Välj **Money** -ikonen (utbetalning) i det övre högra hörnet. Du kan sedan välja andra alternativ, till exempel **transaktions historik** , **betalningar** , **Exportera data** eller **rapporten Office Sales** .

Följande sidor beskrivs här:

- Översikts sidan mina incitament
- Sidan betalningar
- Sidan transaktions historik
- Sidan exportera data och andra associerade export sidor
- Rapport över transaktions Sammanfattning

>[!NOTE]
>Mer information om stimulans-och stimulans funktioner i Partner Center finns i [partner investeringar och incitament](https://partner.microsoft.com/membership/partner-incentives) (inloggning krävs).

## <a name="transaction-history-page"></a>Sidan transaktions historik
På den här sidan visas alla dina enskilda intäkter, inklusive datum, typ och för var och en. Du kan välja en tids period att visa och du kan också filtrera efter registrerings-ID, program, betalnings-ID, typ, Laffont och status. Data är tillgängliga för innevarande räkenskapsår (1 juli 30 juni) och de föregående två räkenskapsåren.

Om du vill se mer information om ett som du väljer väljer du nedåtpilen till höger på sidan. På så sätt visas spaken, intäkts belopp, produkter och kund. Om någon av dessa data inte är tillgänglig, men du behöver åtkomst till den, kan du kontakta supporten. Om resultatet är resultatet av en justering, och inte en transaktion, kommer fälten produkt och kund inte att visas.

Om du vill exportera någon av transaktions data på den här sidan väljer du **Exportera** och följer sedan anvisningarna på sidan Exportera data. Filer som exporteras från sidan transaktions historik visar data i transaktions valuta, intäkter i både transaktions valuta och US-dollar och det betalda värdet i betala till valuta.

### <a name="estimated-payment-month"></a>Beräknad betalnings månad

Sidan transaktions historik innehåller nu en tabell som visar de uppskattade betalnings beloppen för de kommande månaderna. Du kan också visa och hämta informationen i exporten av transaktions historik och sammanfattnings rapporter. Den här informationen gör avstämningar och betalnings projekt enklare.

Den uppskattade betalnings månaden beräknas utifrån program konfigurations regler och tids linjer och bearbetas i nästa/kommande betalnings cykel.

Den uppskattade betalnings månaden är för närvarande tillgänglig för alla typer av typer förutom samop, som visas som **ej tillämpligt** . Den uppskattade betalnings månaden visas som **ej tillgänglig** för intäkter före den 1 juli 2020.

#### <a name="estimated-payment-month-example"></a>Exempel på Beräknad betalnings månad

| Månad | Belopp |
| ------ | :-----------: |
|  Sep-2020 |  $7 273,99   |
|  Okt-2020 | $8 692,30  |
|  Nov-2020 | $107,89  |

Det beräknade beloppet kan variera från den faktiska mängden av olika orsaker:

- Tilldelad omberäkning: om intäkter beräknas om, kommer den faktiska mängden att vara olika
- Justeringar: den faktiska mängden varierar beroende på justeringar som har inträffat eller skickats.
- Regel ändring: en ändring i reglerna kan återspegla omberäkningen av faktiskt betalt belopp
- Debiterad: om betalnings fel inträffar kan den faktiska mängden vara olika

Observera att din betalning endast lanseras under den beräknade månaden om programmets tröskel och regler för betalnings kvalifikationer är uppfyllda. Dessa regler inkluderar, men är inte begränsade till listan nedan:

- Din skatte profil måste vara aktuell
- Dina intäkter måste uppfylla eller överskrida det lägsta gräns värde som definierats i din program guide.
- Utbetalning väntar: om du väljer alternativet "Behåll min betalning" på sidan profil tilldelning.
- Utbetalnings instrument inte tillgängligt: betalnings-eller/och skatte profil har inte slutförts.

## <a name="export-data-page"></a>Sidan exportera data
Följ instruktionerna på den här sidan för att exportera de data du vill ha.

Några saker att tänka på när du försöker exportera data:

- Med tanke på den stora mängden data är service avtalet för den skickade begäran att bearbetas 24 timmar.

- Du ser bara data för MPN-ID: n och program som du är associerad med. Om du vill ha åtkomst till ytterligare data kan du läsa mer i stimulans administratören.

- Sidan exportera data uppdateras inte på egen hand. Du kan behöva uppdatera sidan manuellt för att se de senaste data.

- Filtret kan resultera i att det **inte finns några tillgängliga data** fel. Det här felet innebär förmodligen att du har lämnat standard tids perioden som valts vid tre månader och sedan valt ett betalnings-ID från ett som ligger utanför den perioden. Expandera din tids period och försök igen.

## <a name="transaction-summary-report"></a>Rapport över transaktions Sammanfattning

>[!NOTE]
>Rapporten transaktions Sammanfattning visar inte alla dina program eller intäkter. Det är en sammanfattning som bara fokuserar på aggregerade spaken. 

Rapporten transaktions sammanfattning innehåller transaktions information om sammanfattnings nivå, inklusive kundnamn och prenumerations-ID för sammanställda-handtag. Rapporten innehåller varje ID som är kopplat till flera transaktioner. De totala vinst beloppen och intäkts detaljerna för detta ID upprepas för varje kund/prenumeration. Detta beror på att incitament för sammanställda spaken tas i kollektivt, inte per prenumeration.

4. Välj lämpliga filter om du vill se vilka program och funktioner som är berättigade till sammanlagda intäkter. Om inga data visas är dina program inte berättigade till aggregerade spaken.
5. Klicka på **Starta hämtning**   för att exportera rapporten.  

>[!NOTE]
>Rapporten transaktions sammanfattning innehåller inte kund-och prenumerations information för prenumerationer som är nyare än 60 dagar, eller om förbruknings intäkterna är mindre än en procent.
