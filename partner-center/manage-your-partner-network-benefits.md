# <a name="manage-your-microsoft-partner-network-membership-benefits-and-offers-in-partner-center"></a>Hantera dina Microsoft Partner Network medlemskaps förmåner och erbjudanden i Partner Center

**Lämpliga roller**

- Global administratör
- Administratör för användar hantering
- Administratörs agent
- MPN-partner administratör

>[!IMPORTANT]
>Den här artikeln förklarar hur du börjar använda dina Microsoft Partner Network (MPN) medlemskap i Partner Center – den här webbplatsen. Om du inte har fått någon inbjudan att flytta företags informationen från partner medlemskaps Center (PMC) till partner centret bör du fortsätta att hantera dina MPN-medlemskap i PMC. [Logga in](https://partner.microsoft.com/_login?authType=OpenIdConnect) på PMC nu.

När du ansluter till Microsoft Partner Network (MPN) får du till gång till medlemskaps förmåner som kan hjälpa dig att bygga och utveckla din verksamhet. Om du inte redan är medlem i MPN kan du ansluta till MPN- [webbplatsen](https://partner.microsoft.com/membership).

## <a name="manage-your-membership-benefits-in-the-partner-center"></a>Hantera dina medlemskaps förmåner i Partner Center

När du har köpt ett medlemskaps förmåns paket måste du hämta program varu nycklar och tilldela licenser, prenumerationer och andra fördelar för anställda för att börja använda dem.

>[!IMPORTANT]
>Även om du inte behöver vara global administratör i din organisation för att köpa förmåner, måste du vara global administratör för att tilldela förmåner till anställda. Om det behövs kan du bli global administratör genom att följa anvisningarna på [sidan **förmåner**](https://partnercenter.microsoft.com/pcv/partnership/benefits) (inloggning krävs). Företaget kanske redan har en global administratör för att kunna kontakta ditt team.

Förmåns paket omfattar allokering av licensierad program vara, moln tjänster, Visual Studio-prenumerationer och teknisk support. Förmåns paket kan också omfatta vissa resurser för go-to-marknad, erbjudanden och marknads relaterade program. Du kan få Action Pack prenumerations information på [Microsoft Partner Network](https://partner.microsoft.com/membership/internal-use-software) webbplatsen.  

Om du vill visa och hantera alla dina medlemskaps förmåner går du till [sidan **förmåner**](https://partnercenter.microsoft.com/pcv/partnership/benefits) (inloggning krävs). Alternativt kan du logga in på Partner Center- [instrumentpanelen](https://partner.microsoft.com/dashboard), välja alternativet [**förmåner**] på menyn till vänster och sedan välja eventuella förmåns alternativ som visas.  

För att aktivera de flesta förmåner eller hämta relaterade program varu nycklar kräver Partner Center vanligt vis att någon har tilldelats rollen som global administratör eller MPN partner administratör. Om du inte kan se några förmåner eller inte kan aktivera förmåner kan du kontakta din partner Center Global-administratör för att se till att du har rätt behörigheter. [Läs mer om Partner Center-roller](permissions-overview.md).

## <a name="manage-software-benefits"></a>Hantera program varu förmåner

1. Välj **program vara** om du vill se en lista över program varu förmåner.

2. Hitta den produkt som du vill börja använda och välj sedan nedåtpilen längst till höger på raden för att expandera produktens information.

3. För varje program produkt som du vill börja använda:

   a. Välj **Hämta nycklar** för att Visa produkt licens nycklarna. Se till att följa alla särskilda instruktioner som anges.

   b. Välj det språk som du vill att produkten ska visas i.

   c. Välj operativ system typ (OS) för de datorer som du planerar att installera program varan på.

   d. Följ länkarna för att ladda ned och installera paket för program varu installation.

## <a name="azure-and-cloud-products"></a>Azure-och moln produkter

På fliken **Azure & Cloud Products** visas de användnings prenumerationerna (Azure-krediten) och de licensbaserade prenumerationerna (till exempel Office 365) som är etablerade för dig, licens antal för varje produkt, erbjudande typ som etablerade den förmåns produkten till dig, förmåns typen (oavsett om den kommer från kärna eller stegvis) och förfallo datum för förmånen.

:::image type="content" source="images/benefits/azuretab.png" alt-text="Fliken Azure-och Cloud-produkter":::

### <a name="usage-based-subscriptions---microsoft-azure-credit"></a>Användning-baserade prenumerationer – Microsoft Azure kredit

Det finns två typer av användnings prenumerationer som du kan få, beroende på vilken typ av erbjudande du har aktiverat som MPN-partner – kartor, silver och/eller guld – månatlig kredit eller Mass kredit.

Du kan tillhandahållas med:

- $100/månad (eller motsvarande i lokal valuta) Azure månads kredit eller/och
- $6 000, $12 000 eller/och $13000/år (eller var och en av dem motsvarande i lokal valuta) Azure Mass kredit

Om du vill aktivera en Microsoft Azure-prenumeration i Partner Center tilldelar du åtkomst till en användare i företaget (inklusive gäst användare) som har en användar post som skapats i din Azure Active Directory.
För den $100/månad (eller motsvarande i lokal valuta) Azure månads kredit, kan du när som helst lägga till eller ta bort användare från den här prenumerationen. 

För Azure bulk (årlig) kredit är alternativet ta bort inte tillgängligt i Partner Center, men du kan överföra Mass krediten till någon annan under ditt registrerings år. Kredit prenumerationen för Azure bulk (år) kan inte kombineras med andra erbjudanden för licens, förbetald eller kredit (det vill säga Microsoft Enterprise-avtal, Cloud Solution Partner, Azure i Open, Azure-pass, Action Pack osv.).

## <a name="license-based-subscriptions"></a>Licensbaserade prenumerationer

I avsnittet licensbaserade prenumerationer på fliken Azure & Cloud Products visas alla moln produkter som kräver en produkt nyckel för aktivering, till exempel Microsoft Office 365, Microsoft Dynamics, Microsoft Intune, Azure AD Basic och Power BI. Den typ av erbjudande som du har aktiverat som MPN-partner – kartor, silver eller guld-styr vilka produkter som listas.

Om du vill aktivera licensbaserade prenumerationer klickar du på den nedrullningsbara symbolen i linje med produkten av intresse och läser anvisningarna om hur du löser in nycklarna. När du har läst instruktionerna väljer du den länk som omdirigerar dig till portalen för att lösa in nycklar och den tilldelade nyckeln för produkten.

>[!IMPORTANT]
>Observera att när en nyckel har lösts kan inte samma nyckel lösas igen en andra gång, så se till att du är klar på om du behöver lägga till fler licenser (platser) eller förlänga slutdatumet. Om du väljer att lägga till fler licenser och inte utökar slutdatum kommer systemet att förse dig med färre än 12 månaders aktiva prenumerationer. Kontrol lera att du är inloggad med rätt autentiseringsuppgifter för att aktivera nycklarna på rätt klient organisation.

## <a name="manage-visual-studio-subscriptions"></a>Hantera Visual Studio-prenumerationer

1. Välj **Visual Studio-prenumerationer** för att se listan med Visual Studio-prenumerationer.

2. Hitta den prenumeration som du vill börja använda och välj sedan nedåtpilen längst till höger på raden för att expandera prenumerationens information.

3. Om du vill aktivera en Visual Studio-prenumeration väljer du en användare i listan och väljer sedan **tilldela användare**.

   > [!IMPORTANT]  
   > Om den användare som du vill tilldela inte finns i listan kan du [lägga till nya användare](create-user-accounts-and-set-permissions.md) i **konto inställningar**.

4. Upprepa den här processen för varje prenumeration som du vill tilldela. Användare kan hantera sina prenumerationer i Visual Studio-portalen.

   >[!Note]
   >Prenumerations administratörer för Visual Studio kan omtilldela prenumerationer från en prenumerant till en annan om 90 dagar har passerat sedan tidpunkten för den ursprungliga tilldelningen. Mer information finns i [guiden för Microsoft partner program](https://aka.ms/partner-benefits-use-guide).

## <a name="manage-technical-benefits"></a>Hantera tekniska förmåner

I det här avsnittet beskrivs hur du aktiverar och hanterar tekniska support förmåner på Partner Center- [instrumentpanelen](https://partner.microsoft.com/dashboard). Om ditt Microsoft Partner Network-medlemskap fortfarande finns i partner Membership Center (PMC) måste du dock kontakta en annan resurs som partner Frontline för att hjälpa dig att aktivera dina tekniska förmåner. Se [Aktivera tekniska förmåner utanför Partner Center](partner-membership-center-tech-benefits-activate.md).

Gör så här om du vill se dina tekniska förmåner i Partner Center:

1. Logga in på [instrument panelen](https://partner.microsoft.com/dashboard)för partner Center.

2. Välj **förmåner** och sedan **tekniska fördelar** från den vänstra navigerings menyn. Sidan tekniska förmåner visas med en lista med tekniska support förmåner som är tillgängliga för alla dina guld-, silver-eller Microsoft Action Pack-kompetenser.

   Om du inte ser de här alternativen eller sidan tekniska förmåner kanske du inte har tilldelats rätt [användar roller](permissions-overview.md) i Partner Center. Kontakta din partner Center Global-administratör för att få åtkomst.

3. Expandera listan över förmåner. För att komma igång med en speciell support förmån måste du göra följande första gången du använder den:

   - **Aktivera den tekniska fördelen.** När du har expanderat en förmånens information väljer du **Aktivera**. Tillåt några dagar för att bearbeta din aktiveringsbegäran.

   - **Visa och dela support åtkomst-ID och kontrakt-ID.** När din aktiveringsbegäran har bearbetats visas information om **åtkomst-ID** och **kontrakt-ID** nästa gång du expanderar den tekniska fördelen. Du eller ditt företags användare kan behöva dela den här informationen första gången du skickar en support förfrågan för vissa typer av produkter eller tjänster.

4. När den har Aktiver ATS kan du använda tekniska förmåner för olika typer av teknisk support. Om du vill skicka tekniska support förfrågningar till följande tjänster läser du de detaljerade anvisningarna på sidan tekniska förmåner. Se även informationen nedan.

   Sidan med tekniska förmåner för partner Center beskriver dessa typer av teknisk support:

   **Teknisk support (bryta-Fix).** Använd den här typen av teknisk support när en funktion är bruten eller om något inte fungerar som förväntat (till exempel när du får ett fel meddelande). Här är några rikt linjer:

   - Använd **Signature Product Support** för **moln produkter** endast för guld/silver-kompetens partner (inte tillgänglig för Action Pack prenumeranter).

   - För både Action Pack-och guld-och silver kompetens partners använder du **Microsofts produkt support** incidenter för **senaste lokala produkter** (endast aktuella och tidigare versioner).

     >[!NOTE]
     > Åtkomst-ID och kontrakt-ID krävs för tekniska support förfrågningar för lokala, Azure-och Dynamics-produkter. För att hitta ditt åtkomst-ID och kontrakt-ID kan du också läsa [hitta dina förmåner](mpn-find-benefits.md#find-access-id-and-contract-id-for-technical-benefits).
  
   **Tekniska försäljnings-och distributions tjänster.**  Använd den här typen av support för samråd eller råd om teknisk försäljning eller distribution. Dessa tjänster kan begäras direkt från [partner Center](https://partnercenter.microsoft.com/pcv/partnership/benefits/createadvisoryhoursservicerequest).

   - För både Action Pack-och guld-och silver kompetens partners använder du **tekniska försäljnings-och distributions tjänster** för moln-och hybrid lösningar.
  
## <a name="manage-go-to-market-offers"></a>Hantera go-to-Marketing-erbjudanden

**Lämpliga roller för att utföra den här uppgiften:**

- Global administratör
- MPN-partner administratör

1. Välj **förmåner** och sedan **gå till marknaden** för att se en lista över erbjudanden som är associerade med go-to-Marketing med Microsoft-program.

   [Funktionerna för att gå till marknaden](mpn-learn-about-go-to-market-benefits.md) erbjuder ytterligare marknadsförings resurser, erbjudanden, program och hjälp som kan vara tillgängliga för dig. Vilka typer av erbjudanden som är tillgängliga beror på din partner nivå och alla befogenheter som du kanske redan har fått.

2. Om du vill aktivera valfritt erbjudande eller program för go to Marketing måste du först tilldela en företags marknadsförings kontakt. Den här kontakten får uppföljnings kommunikation om ditt aktiva erbjudande till marknaden.

3. Om du vill lägga till eller uppdatera din marknadsförings kontakt information går du till början av sidan gå till marknaden och väljer sedan **Lägg till**, **Uppdatera** eller **ändra**. Nu ska du utföra följande steg:  

   a. Välj en användare i listan. Om den användare som du vill tilldela inte finns i listan kan du [lägga till nya användare](create-user-accounts-and-set-permissions.md) i **konto inställningar**.

   b. Ange en e-postadress för den användare som skiljer sig från den e-postadress som är kopplad till ditt företags Partner Center-konto.

      Vi kommer att få anvisningar om hur du använder ditt go-to-Marketing-erbjudande till din utsedda marknadsförings kontakts e-postadress.

   c.  Ange telefonnummer och språk för den här marknadsföringskontakten. När du har angett den här informationen väljer du **Tilldela användare**.

4. När du har uppdaterat din marknadsförings kontakt letar du reda på det erbjudande till marknaden som du vill börja använda och väljer sedan nedåtpilen längst till höger på raden för att expandera erbjudandets information. Listan visar varje typ av erbjudande som är tillgängliga under förmåns namnet (till exempel **silver core**, **Gold Core** eller **ISV Co-försäljningen**).

5. Välj **Aktivera** för att aktivera erbjudandet.

   > [!NOTE]
   >Det kan ta flera dagar att skicka instruktioner till den tilldelade marknadsförings kontakten och aktivera ett erbjudande till marknads erbjudande. När erbjudandet har Aktiver ATS visas ändringarna på sidan gå till marknaden. I kolumnen **status** visas erbjudandet Change to **Active**. I kolumnen **upphör ande** visas det datum då erbjudandet upphör att gälla. Du kan använda det här erbjudandet fram till dess förfallo datum. Lär dig mer om de [olika typerna av Go-to-Marketing-resurser, erbjudanden och program](mpn-learn-about-go-to-market-benefits.md) som är tillgängliga.  

## <a name="cancel-a-membership-benefit-or-offer"></a>Avbryta en medlemskaps förmån eller ett erbjudande

För att kunna avbryta en förmån eller erbjuda och få en åter betalning måste du göra en begäran inom 30 dagar från köpet och inte ha aktiverat fördelarna med program varan. Begär uppsägningen genom en tjänstbegäran.
